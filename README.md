# 🩺 OrthoVision
AI-based early detection of scoliosis using TensorFlow, Mediapipe and OpenCV

---

## 🚀 Overview
OrthoVision is a deep learning project designed to assist in the early detection and severity analysis of scoliosis using computer vision techniques.  
It leverages **TensorFlow**, **Mediapipe**, and **OpenCV** to process posture images and generate meaningful insights.

---

## ✨ Features
- 📷 Pose landmark detection using Mediapipe  
- 🧠 Deep learning models (TensorFlow/Keras) for classification  
- 📊 Visualization of training metrics and ROC curves  
- 🖼️ Output images for severity distribution and intermediate predictions  
- 🔍 Modular design for easy experimentation  

---

## 🛠️ Tech Stack
- **Languages:** Python  
- **Libraries:** TensorFlow, Mediapipe, OpenCV, Scikit-learn, Pandas, Matplotlib, Seaborn  
- **Tools:** Jupyter Notebook, VS Code, GitHub
